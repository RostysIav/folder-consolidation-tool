WHAT THE SCRIPT DOES STEP BY STEP
═════════════════════════════════════════════════════════════════════════════

1️⃣ READS SOURCE FOLDERS
   From: C:\Users\John\Documents
   From: D:\Backup_2019
   From: E:\Archive

2️⃣ CHECKS DESTINATION
   If C:\Consolidated_Files doesn't exist → Creates it

3️⃣ FOR EACH FOLDER IT ENCOUNTERS:

   If folder "MyProjects" doesn't exist in destination:
     ✓ Create it and copy all files

   If folder "MyProjects" already exists in destination:
     ✓ Rename new one to "MyProjects_2"
     ✓ Copy all files to "MyProjects_2"

4️⃣ FOR EACH FILE IT ENCOUNTERS:

   If file doesn't exist in destination:
     ✓ Copy it

   If file exists and is IDENTICAL (same content):
     ✓ Skip it (saves storage space!)

   If file exists but is DIFFERENT:
     ✓ Rename it: "document.pdf" → "document_2.pdf"
     ✓ Copy it (keeps both versions)

5️⃣ LOGS EVERYTHING
   Detailed report of all actions taken

6️⃣ DONE!
   You have one consolidated folder with no data loss


═════════════════════════════════════════════════════════════════════════════
FILE COMPARISON (IDENTICAL vs DIFFERENT)
═════════════════════════════════════════════════════════════════════════════

The script uses MD5 hashing to check if files are truly identical:

IDENTICAL FILES (Same content):
  report.txt (from C:)     → MD5: a1b2c3d4e5f6g7h8
  report.txt (from D:)     → MD5: a1b2c3d4e5f6g7h8

  Action: Keep only one copy ✓
  Result: Saves storage space!

DIFFERENT FILES (Same name, different content):
  report.txt (from C:)     → MD5: a1b2c3d4e5f6g7h8
  report.txt (from D:)     → MD5: x9y8z7w6v5u4t3s2

  Action: Keep both, rename second to "report_2.txt"
  Result: No data loss ✓
